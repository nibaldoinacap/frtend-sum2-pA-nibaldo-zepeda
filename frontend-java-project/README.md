# ViajarYa - Agencia de Turismo

Sitio web frontend para la agencia de viajes "ViajarYa". El proyecto consta de:
- Una página principal (`index.html`) con el catálogo de destinos.
- Un formulario interactivo (`reserva.html`) para solicitar paquetes turísticos.
- Validaciones completas del lado del cliente utilizando JavaScript puro.
